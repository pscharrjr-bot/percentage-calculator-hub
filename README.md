## Included pages
- Home page
- Percentage Calculator
- Percentage Increase Calculator
- Percentage Decrease Calculator
- Percent Change Calculator
- What Percent Of Calculator
- Discount Calculator
- Markup Calculator
- Margin Calculator
- Privacy Policy

## Publish to GitHub Pages
1. Upload all files in this folder to your GitHub repo root.
2. Commit the changes.
3. In GitHub, open the repo settings.
4. Go to **Pages**.
5. Set source to deploy from the main branch root.
6. Save.
7. Your site should appear at:
   `https://pscharrjr-bot.github.io/percentage-calculator-hub/`
